https://abraaorosal.github.io/CPRAIO_CPJM/

## Estado da publicação

O site está pronto no repositório, com os arquivos principais na raiz e também com a branch `gh-pages` criada para compatibilidade com o modo clássico do GitHub Pages.

Para ativar definitivamente pelo GitHub Pages, use:

1. Settings do repositório;
2. Pages;
3. Source: Deploy from a branch;
4. Branch: `gh-pages`;
5. Folder: `/root`;
6. Save.

Após salvar, o endereço acima passa a funcionar em alguns minutos.
